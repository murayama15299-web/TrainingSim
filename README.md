# TrainingSim

Unityで育成システムの基礎を理解するために制作したプロトタイプです。

## 概要
ターン制でトレーニングを選択し、ステータスを成長させて最終評価を目指す育成ゲームです。
成功・失敗・大成功といった確率要素と、体力管理による意思決定を含んでいます。

## 操作方法
- 各トレーニングボタンをクリックして実行
- 「お休み」で体力回復
- 規定ターン終了で最終評価表示

## 制作目的
企画者として、以下を自分の体感として理解することを目的に制作しました。

- 仕様設計と実装コストの関係
- 数値調整によるゲーム体験の変化
- UI更新とゲームロジックの連携

実際にUnity上で実装・調整することで、
「この仕様は重い」「この変更は簡単にできる」といった感覚を掴むことを重視しています。

## 使用技術
- Unity
- C#
- TextMeshPro
